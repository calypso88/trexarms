disable()method 
public static function disable(traceMessagesWhileDisabled:Boolean = true):void

Empties stored data and turns off the Console. Any calls to logging functions after the Console is disabled will return immediately. This should be used when deploying release builds.

Parameters

traceMessagesWhileDisabled:Boolean (default = true) — If true, disabled logging calls will be passed to the trace() function.
